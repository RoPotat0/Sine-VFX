Install

Four steps

  1. Get the plugin

    Install from the Creator Store, or download the loader from the repo into your Studio Plugins folder.

    Open the repo

  2. Allow HTTP once

    Studio asks for permission the first time SineVFX contacts the update server. Accept it — updates and licence checks both go through it.

  3. Open it

    Click SineVFX in the Plugins tab. Select any emitter, beam or trail and the properties panel fills in.

  4. It updates itself

    Every launch pulls the newest release automatically. No reinstalling.
